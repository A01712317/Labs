Diferencia entre Internet y la World Wide Web

WWW es un servicio al que podemos acceder mediante internet, ya que internet es una herramienta que ayuda a navegar sitios www.

Partes de un URL
  • https (protocolo)
  • www (subdominio)
  • nombre.com (dominio)

Métodos HTTP
  • GET: Se solicita un dato.
  • HEAD: Brinda información de un dato, pero sin obtenerlo.
  • POST: Envía los datos de un recurso para interactuar con ellos.
  • PUT: Reemplaza o actualiza todo el contenido de un recurso ya existente.
  • PATCH: Reemplaza o actualiza datos específicos, no todos.
  • DELETE: Elimina un recurso.

Método HTTP para formularios HTML

Se debe utilizar POST, ya que brinda cierta seguridad al limitar la visibilidad de los datos personales.

Método HTTP al acceder a una página

Se utiliza GET, ya que accede a los datos solicitados sin modificarlos.

Código de respuesta HTTP 200

Significa que la solicitud ha sido procesada exitosamente. No ocurrió ningún error.

Errores 404 y 500

Error 404: Es responsabilidad del desarrollador corregirlo, ya que algunas rutas pueden estar mal definidas.

Error 500: También es responsabilidad del desarrollador, ya que debe ser un problema en el servidor.

Atributos HTML5 depreciados

HTML5 recomienda usar atributos de CSS para realizar acciones como cambiar el color de texto, centrar contenido, definir color en tablas, alinear contenido.

Diferencias entre HTML4 y HTML5
  • HTML5 incluye elementos que mejoran la experiencia y reemplazan otros métodos que usaba HTML4, como el uso de div. Ahora existen elementos como header, footer, article, section, nav, main, etc.
  • Puede incluir archivos de audio y video sin necesidad de plugins externos.
  • Ofrece más tipos de entrada como email, fecha, color, etc.
  • Integra APIs para funciones que permiten acceder a geolocalización, canvas, etc.

Componentes de estructura y estilo de una tabla
  • Table
  • Table Row
  • Table Header
  • Table Data
  • Thead: Grupo de filas de encabezados.
  • Tbody: Agrupa datos, como celdas de una columna.
  • Table Foot: Agrupa los pies de una tabla.
  • Estilos: Borders, padding, margin, background-color, text-align.
